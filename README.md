**Цель проекта:**



Разработка системы для отображения свойств пласта в межскважинной области на основе данных геологоразведки и предсказаний.



**Основные задачи:**

* Анализ данных – изучение предоставленных геологических данных
* Разработка архитектуры – создание модульной системы обработки и визуализации
* Реализация интерфейса – создание инструментов для 3D-визуализации и анализа
* Демонстрация – запись демонстрации возможностей системы
* Документация – подготовка презентации и технической документации
